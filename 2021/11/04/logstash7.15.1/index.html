Centos7安装ELK日志系统之logstash7.15.1

发表于 2021-11-04 分类于 ELK ， logstash 阅读次数：

一、下载logstash-7.15.1-x86_64.rpm安装包

下载地址: https://www.elastic.co/cn/downloads/logstash

二、将文件拷贝至服务器/usr/local 目录下

三、安装

四、修改配置文件

这里以服务器IP192.168.17.133为例,根据实际情况修改 修改内容如下:

更改前 更改后
beats {
        port => 5044
} 		kafka {
        #（kafka的IP和端口）
        bootstrap_servers => "192.168.17.133:9092"
        client_id => "test1"
        # （消费者分组，可以通过组 ID 去指定，不同的组之间消费是相互不受影响的，相互隔离）
        group_id => "test1"
        auto_offset_reset => "latest"
        # （消费者线程个数）
        consumer_threads => 5
        # （在输出消息的时候会输出自身的信息包括:消费消息的大小， topic 来源以及 consumer 的 group 信息）
        decorate_events => true
        # （主题）
        topics => ["log-topic-test"]
        # （用于ES索引）
        type => "log-topic-test"
        # （数据格式）
        codec => "json"
        tags => ["log-topic-test"]
}
elasticsearch {
        hosts => ["http://localhost:9200"]
        index => "%{[@metadata][beat]}-%{[@metadata][version]}-%{+YYYY.MM.dd}"
        #user => "elastic"
        #password => "changeme"
} 		if "log-topic-test" in [tags] {
        elasticsearch {
                # ES的IP和端口
                hosts => ["192.168.17.133:9200"]
                # 索引名称：主题+时间
                index => "log-topic-test%{+YYYY-MM-dd}"
                timeout => 300
        }
        stdout {
                codec => rubydebug
        }
}

五、加入logstash服务并启动(先启动zookeeper、kafka、elasticsearch)
