记录mysql出现this is incompatible with sql_mode=only_full_group_by问题

发表于 2021-11-25 分类于 mysql ， only_full_group_by 阅读次数：

搭建mysql新环境时原有的业务sql出现sql异常

问题原因

ONLY_FUll_GROUP_BY的意思是：对于GROUP BY聚合操作，如果在SELECT中的列，没有在GROUP BY中出现，
那么这个SQL是不合法的，因为列不在GROUP BY语句中，也就是说查出来的列必须是GROUP BY之后的字段，
或者这个字段出现在聚合函数里面。

mysql5.7版本以后only_full_group_by是默认开启的,查看以下GLOBAL.sql_mode的值

直接在GLOBAL.sql_mode中去掉only_full_group_by(其他值保留)

x为查询到的GLOBAL.sql_mode值去掉only_full_group_by
