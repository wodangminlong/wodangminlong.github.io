Java中的位移运算

发表于 2021-11-30 分类于 java ， java 位移运算

位移运算就是在二进制的基础上对数字进行平移

Java中byte、short、char类型的计算都是先将其转换为int类型,位移也一样。

int类型为32位,所以byte、short、char和int值位移时最多可以位移31位,位移32位时为0,位移33位时和位移1位时得到的结果一样,以此类推。

运算规则

将int值转换为二进制,对转换后的二进制移动对应的位数,高位移除,低位有空位时用0(负数用1)补上。
